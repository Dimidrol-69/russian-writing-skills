# russian-writing-skills

Три скилла для работы с русским текстом, адаптированные для **Codex**. Каждый отвечает за свой слой и не лезет в чужой: их можно использовать по отдельности или собирать в один pipeline.

Изначально создан под Claude. В этой версии скиллы приведены к формату Codex:
- валидный `SKILL.md` frontmatter,
- `agents/openai.yaml` для UI-метаданных,
- совместимая структура папок для установки в `~/.codex/skills`.

| Скилл | Что делает | Слой |
|---|---|---|
| **editor-ru** | Режет воду, чинит логику и порядок, распутывает фразы, уплотняет текст | Структура и смысл |
| **humanizer-ru** | Убирает признаки ИИ-генерации, сохраняя живой авторский голос | Стиль |
| **proofreader-ru** | Правит орфографию, пунктуацию, опечатки и согласование | Механика |

## Зачем это

Один запрос в духе «сделай текст лучше» обычно смешивает три разные задачи:
- структурную редактуру,
- снятие ИИ-отпечатка,
- корректуру.

Если делать всё сразу одним проходом, модель часто начинает либо переписывать лишнее, либо терять авторский голос, либо смешивать стилистику с механикой. Поэтому набор разделён на три скилла: каждый знает свою зону ответственности и не подменяет остальные.

## Pipeline

Логический порядок прогона:

```text
editor-ru  →  humanizer-ru  →  proofreader-ru
структура     стиль            механика
```

1. `editor-ru` приводит в порядок смысл, плотность, переходы и композицию.
2. `humanizer-ru` снимает шаблонный AI-стиль в блоговом тексте.
3. `proofreader-ru` делает финальную вычитку без переписывания голоса.

Каждый скилл при этом самодостаточен. Если нужен только один слой, используй только его.

## Скиллы

### `editor-ru`

Структурный редактор русского текста в оптике ясности и плотности. Убирает воду, схлопывает повторы мысли, чинит переходы, поднимает главное выше и распутывает перегруженные предложения.

Не используется для anti-AI-стилизации и не заменяет корректуру.

Типовые триггеры:
- `отредактируй`
- `сократи`
- `много воды`
- `приведи текст в порядок`
- `сделай яснее`
- `структура хромает`

### `humanizer-ru`

Анти-ИИ-скилл для русского блогового и постового текста. Чистит менторский тон, шаблонные вводные конструкции, LinkedIn-пафос, канцелярит, «идеально ровный» ритм и другие маркеры текста, который звучит как генерация.

Подходит для Telegram-постов, vc.ru, Хабра, заметок, разборов и эссе. Не предназначен для академических, юридических, greentext и коротких reply-текстов.

Типовые триггеры:
- `сделай менее ИИ-шно`
- `перепиши живее`
- `звучит как ChatGPT`
- `слишком сухо`
- `шаблонно`
- `выглядит как LinkedIn-пост`

### `proofreader-ru`

Корректор русского текста без переписывания стиля. Исправляет только орфографию, пунктуацию, опечатки, грамматику, согласование и заглавные буквы.

Сохраняет:
- сленг,
- англицизмы,
- эмодзи,
- Markdown,
- переносы строк,
- типографику автора.

Не лезет в стиль, структуру и anti-AI-правки. Используется как финальный консервативный проход.

Типовые триггеры:
- `проверь на ошибки`
- `вычитай`
- `корректура`
- `поправь пунктуацию`
- `поправь грамматику`
- `опечатки есть?`

## Установка в Codex

### Вариант 1. Клонировать репозиторий и скопировать папки вручную

```bash
git clone https://github.com/Dimidrol-69/russian-writing-skills.git
cp -r russian-writing-skills/editor-ru ~/.codex/skills/
cp -r russian-writing-skills/humanizer-ru ~/.codex/skills/
cp -r russian-writing-skills/proofreader-ru ~/.codex/skills/
```

На Windows это обычно:

```powershell
git clone https://github.com/Dimidrol-69/russian-writing-skills.git
Copy-Item -Recurse russian-writing-skills\editor-ru $HOME\.codex\skills\
Copy-Item -Recurse russian-writing-skills\humanizer-ru $HOME\.codex\skills\
Copy-Item -Recurse russian-writing-skills\proofreader-ru $HOME\.codex\skills\
```

После установки перезапусти Codex, чтобы он подхватил новые скиллы.

### Вариант 2. Через skill-installer в Codex

Можно установить репозиторий прямо из Codex через системный скилл `skill-installer`, если удобнее тянуть с GitHub, а не копировать руками.
